# Substitution_Cipher

This is a simple substitution cipher. 
- It includes the encryption and decryption logic

# What is a substitution cipher?

- This is a type of encryption created by systematically replacing plaintext with other letters, numbers or symbols
- It is simple to make but is insecure for modern communication due to frequency analysis and pattern recognition

# How to use:  
- Save the EncDec.java file on your device
- Open any IDE (VScode was used for this)
- Travel to the directory that your EncDec.java file is in (e.g: cd \Desktop\EncDec.java)
- Compile the code: javac EncDec.java
- Run the code: java EncDec
- Follow the instructions
